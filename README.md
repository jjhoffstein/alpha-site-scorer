# Alpha Site Pipeline

AI-Powered Site Viability Scoring System for rapid school campus evaluation.

## Live Demo

**[View Dashboard](https://jjhoffstein.github.io/alpha-site-scorer/)**

## Features

- **Weighted Scoring** — Evaluates sites on sq footage, driveway access, outdoor space, kitchen facilities, zoning, and cost efficiency
- **Go/No-Go Decisions** — Automated recommendations with 70/100 threshold
- **Pipeline Ranking** — Compare multiple sites instantly
- **Weak Area Detection** — Identifies specific concerns for each site

## Scoring Criteria

| Factor | Weight | Description |
|--------|--------|-------------|
| Square Footage | 30 | Target student capacity (50 sqft/student) |
| Driveway Access | 20 | Controlled arrival/dismissal logistics |
| Outdoor Space | 15 | Play area, basketball, turf |
| Kitchen Area | 15 | Food service infrastructure |
| Zoning | 10 | Clear permitting path |
| Cost Efficiency | 10 | $/sqft ratio optimization |
